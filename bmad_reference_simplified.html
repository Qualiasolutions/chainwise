Claude Code BMAD Method Reference

Build, Manage, Analyze, Deploy - AI Development Framework

🚀 Quick Start Workflow

Planning Phase

  1. *analyst [project description] - Research and analysis
  2. *pm Create requirements document
  3. *architect Design system structure
  4. Review and approve plans

Development Phase

  1. *sm Break into development tasks
  2. *dev Implement features
  3. *qa Test and validate
  4. *security-auditor Security review

🤖 BMAD Agents

*analyst

Market research, feasibility studies, competitive analysis

*pm

Requirements, user stories, project management

*architect

System design, technology selection, architecture

*sm

Sprint planning, task breakdown, progress tracking

*dev

Full-stack development, coding, implementation

*qa

Testing, quality assurance, bug validation

*security-auditor

Security review, vulnerability testing

⚡ Slash Commands

/create-demo [description]

Build complete demo application from description

Use for: Quick prototypes, client demos, proof of concepts

/diagnose [issue]

Debug and fix system issues automatically

Use for: Build errors, dependency problems, performance issues

/enhance [feature]

Optimize and improve existing code/features

Use for: Performance improvements, code refactoring

/remake [feature]

Rebuild feature with modern technologies

Use for: Legacy modernization, framework upgrades

/generate-prp [project]

Create comprehensive project requirements

Use for: Large project planning, client documentation

/execute-prp [file]

Implement complete project from requirements

Use for: End-to-end project delivery
